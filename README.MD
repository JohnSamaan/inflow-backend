# Inflow - Backend

**Inflow** is a robust Supply Chain Management (SCM) system designed to streamline warehouse operations. This backend service manages inventory flow, warehouse organization, and administrative personnel tracking.

## 1. Key Features
* **Warehouse Management:** Track multiple locations and their capacities.
* **Personnel Tracking:** Manage staff roles and access across different hubs.
* **Inventory Control:** Real-time monitoring of goods moving through the supply chain.

## 2. Tech Stack
* **Language:** Java 21
* **Framework:** Spring Boot
* **Database:** PostgreSQL
* **Build Tool:** Maven
* **IDE:** IntelliJ IDEA
